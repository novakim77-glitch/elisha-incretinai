텔레그램 봇과 연결하기 (강력 추천)
앱을 매번 안 열어도, 카톡하듯 자연어로 코칭 받기
🤖 왜 봇이 필요한가요? "오늘 아침 물 마셨어", "지금 73kg" 같이 평소 말하듯 보내면 봇이 자동으로 루틴을 체크하고 체중을 기록합니다. 매일 07:00 / 18:30 / 22:00 KST에 골든타임 알림 · 메타볼릭 스위치 콜 · 일일 리캡도 자동으로 보냅니다.
1
텔레그램에서 봇 검색 + 시작
텔레그램 앱 검색창에 @incretina_ai_bot 입력 → 봇 선택 → 하단 "시작" 탭.
🔍 또는 직접 열기 → t.me/incretina_ai_bot
↓
2
앱에서 연결 코드 발급
앱 실행 → 우측 상단 프로필 메뉴 → 텔레그램 연결 → 6자리 코드가 5분간 표시됩니다.
↓
3
텔레그램에서 코드 입력
텔레그램 봇 채팅창에 /link 123456 형식으로 입력 (123456 자리에 발급받은 본인 코드). 성공하면 "🎉 IncretinA i 계정과 연결되었어요!" 메시지가 옵니다.
↓
4
자연어로 사용 시작
이제 봇에 평소 말하듯 보내면 됩니다. 예시:
💬 "오늘 아침 물 마셨고 30분 걸었어" → 루틴 자동 체크
💬 "오늘 73.2" → 체중 자동 기록
💬 "오늘 점수 어때?" → IMEM α/β/γ 분석
💬 "지금 골든타임이야?" → 현재 상태 답변
🎭 코칭 톤 변경: 봇 채팅창에 /persona empathetic (따뜻한 코치 · 기본) / /persona clinical (임상 데이터형) / /persona driver (동기부여형) 중 선택할 수 있습니다.

설치 후 이렇게 사용하세요
앱을 최대한 활용하는 방법
  • 프로필 설정: 이름, 키, 체중, 목표 체중을 입력하면 맞춤 분석이 시작됩니다
  • 텔레그램 봇 연결 (강력 추천): 위 가이드대로 봇과 연결하면 자연어로 모든 기록이 가능합니다
  • 매일 루틴 체크: 앱에서 원터치로 또는 봇에 "X 했어"라고 보내기
  • 골든타임 활용: 일출~일몰 시간에 식사와 활동을 집중하세요
  • 자동 알림: 봇이 매일 07:00 (모닝) / 18:30 (메타볼릭 콜) / 22:00 (리캡) 자동 발송
  • 주차별 잠금 해제: 주차가 올라갈수록 더 많은 루틴이 해제됩니다
  • 배지 수집: 프로필 탭에서 획득한 배지를 확인하고 모든 배지를 모아보세요

자주 묻는 질문
💬 앱 스토어에서 검색해도 안 나와요
IncretinA i Pro는 웹 앱(PWA)이라 앱 스토어에 등록되어 있지 않습니다. 위의 링크로 접속하여 홈 화면에 추가하면 동일하게 사용할 수 있습니다.
💬 데이터가 날아갈까 걱정돼요
모든 데이터는 Firebase 클라우드에 자동 저장됩니다. 기기를 바꿔도 같은 이메일로 로그인하면 데이터가 그대로 복원됩니다.
💬 가족/지인과 함께 쓸 수 있나요?
네! 각자 다른 이메일로 가입하면 개별 계정으로 사용할 수 있습니다. 같은 기기에서도 로그아웃 후 다른 계정으로 로그인 가능합니다.
💬 인터넷 없이도 사용할 수 있나요?
기본적인 루틴 체크는 오프라인에서도 가능합니다. 인터넷에 연결되면 자동으로 데이터가 동기화됩니다.
💬 텔레그램 봇과 앱 데이터는 같은가요?
네, 동일합니다. 봇이 기록한 루틴/체중은 즉시 앱에도 반영되고, 앱에서 한 기록도 봇이 그대로 인식합니다. 같은 Firebase 계정으로 양방향 동기화됩니다.
💬 봇을 꼭 사용해야 하나요?
필수는 아니지만 강력 추천합니다. 앱을 매번 열지 않아도 카톡하듯 한 줄만 보내면 자동으로 기록되고, 자동 알림으로 골든타임을 놓치지 않게 도와줍니다.
💬 봇이 제 데이터를 다른 사람과 공유하나요?
아닙니다. 봇은 본인의 텔레그램 계정과 본인의 IncretinA i 계정만 1:1로 연결합니다. 다른 사용자와 절대 공유되지 않으며, 모든 대화는 본인 Firestore에만 저장됩니다.
