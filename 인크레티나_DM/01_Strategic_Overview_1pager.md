# 인크레티나 (IncretinA)
## *내 몸이 만드는 GLP-1으로, 약 없이 대사를 회복한다*

> **One Engine, Two Indications.**
> 비만(인크레티나 i) → 당뇨(IMEM-DM 신규 브랜드)로 확장하는 인크레틴 자가분비 디지털 헬스 플랫폼

---

### 🌍 시장 기회 (Why Now)

GLP-1 약물 시장이 폭발하며 **"내 몸의 GLP-1을 깨우는 디지털 솔루션"**이 새 카테고리로 부상.

| 시장 신호 | 의미 |
|---------|------|
| GLP-1 약물 시장 **2030년 1,500억 달러+** (모건스탠리) | 사용자 인식·언어 형성 완료 |
| 약물 사용자 **연 7-10% 중단** (가격·부작용·자가주사 부담) | 라이프스타일 대안 수요 폭발 |
| CGM(연속혈당측정) 보급 확대 + 디지털 치료기기(DTx) 규제 성숙 | 기술-규제 환경 동시 도래 |
| 한국 당뇨 인구 600만, 전당뇨 1,500만 | 국내 시장만으로 검증 가능한 규모 |

### 💡 솔루션 — IMEM 엔진

**IMEM (Incretin Metabolic Efficiency Model)**: 인크레틴 호르몬 분비 효율을 정량화하는 행동 과학 기반 알고리즘.

```
대사 효율 = α (일주기 리듬) × β (영양 시퀀스) × γ (신체 활동)
```

- **α**: 식사·수면·광노출 시간 정렬 → 인크레틴 분비 리듬 동조
- **β**: 채소→단백질→탄수 시퀀스 → L-cell GLP-1 직접 유도
- **γ**: 식후 보행, 근력 운동 → GLUT4·인슐린 감수성 회복

**과학적 근거**: Vilsbøll et al. (2003), Kuwata et al. (Diabetologia 2016), Shukla et al. (Diabetes Care 2015) 외 다수 RCT.

**구현체**:
- PWA 앱 (Firebase + Chart.js, GitHub Pages 배포)
- 텔레그램 AI 코치 봇 (Claude Sonnet 4.5, Fly.io 운영)
- 실시간 IMEM 계수 + Bio-Sync 골든타임 카운트다운

| **기술 스택** | PWA (오프라인 가능), Firestore 실시간 동기화, Claude API 자연어 코칭 |

> 💬 **사용자 패턴**: 텔레그램 봇이 진입 장벽을 낮춰 일일 평균 5-7회 자연스러운 대사 코칭 대화 발생.
> 앱-봇 데이터 양방향 동기화로 "기록 앱"이 아닌 "대화하는 코치" 경험 제공.

### 🧬 플랫폼 비전 — **Dual-Brand 전략**

제약 GLP-1 듀얼 브랜드 모델(노보 노디스크의 **Ozempic ↔ Wegovy**, 일라이 릴리의 **Mounjaro ↔ Zepbound**)을 디지털 헬스에 재현.

```
       IMEM Engine (공통 코어)
                │
    ┌───────────┴───────────┐
    ▼                       ▼
인크레티나 i              [당뇨 브랜드]
(비만 — 현재)             (당뇨 — 확장 계획)

  IMEM 캘리브레이션          IMEM-DM 캘리브레이션
  포만감·체중 중심           혈당 평탄·인슐린 감수성 중심
```

**한 엔진, 두 적응증, 두 브랜드.** 같은 분자(semaglutide)가 Ozempic(당뇨)과 Wegovy(비만)로 분리되어 각기 시장을 잡듯, **IMEM 엔진도 적응증별로 캘리브레이션·내러티브·UI를 분기**하여 각 시장에서 정체성·임상 가치 극대화.

**기술 자산 공유율 ~80%**: 코어 엔진·텔레그램 봇·Firestore 인프라·AI 코칭 백엔드 모두 공유. 신규 브랜드 추가 시 한계 비용 최소.
