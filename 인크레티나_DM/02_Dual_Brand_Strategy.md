> **One Engine, Two Indications, Two Brands.**
>
> 본 문서는 인크레티나(IncretinA) 브랜드의 듀얼 적응증(비만 → 당뇨) 확장을 위한 전략 골격입니다.

## 1. Executive Summary

### 1.1 핵심 명제

> **인크레티나는 단일 제품이 아니라 인크레틴 자가분비 활성화 플랫폼이다.**
> 같은 IMEM 엔진을 비만(인크레티나 i)과 당뇨(IMEM-DM 신규 브랜드) 두 적응증으로 분기하여, 제약 GLP-1의 듀얼 브랜드 전략을 디지털 헬스에서 재현한다.

### 1.2 전략의 4가지 기둥

| 기둥 | 내용 |
|------|------|
| ① **공통 코어** | IMEM 엔진(α/β/γ + Bio-Sync), 텔레그램 AI 코치, Firestore 인프라 — 두 브랜드 공유 |
| ② **분기 캘리브레이션** | 적응증별로 계수 가중치·신규 계수·위험 카탈로그 분기 (IMEM vs IMEM-DM) |
| ③ **분기 내러티브** | 사용자 정체성·메시지·UI·KPI 완전 분리 (체중감량 vs 혈당 평탄) |
| ④ **분기 채널** | D2C(비만) vs 의료진 동반(당뇨), 다른 가격대·다른 LTV |

### 1.3 우선순위

- **Phase 1 (현재 ~ +6개월)**: 인크레티나 i (비만) 상용화 100% 집중
- **Phase 2 (+6 ~ +12개월)**: IMEM-DM 특허 출원 + 내부 R&D 트랙 (별도 인력)
- **Phase 3 (+12 ~ +24개월)**: 당뇨 브랜드 베타 → 정식 런칭

> **원칙**: IMEM-DM은 **IR·특허·내부 비전 자산**으로만 활용. 인크레티나 i 핵심 팀의 리소스는 분산되지 않음.

### 1.4 자산 가치 명제

VC 입장에서 본 가치 재평가:

```
[기존] 단일 제품 (비만 디지털 헬스 앱)
   → Multiple: 5-10x revenue

[전환 후] 인크레틴 플랫폼 (비만 + 당뇨 + 향후 적응증)
   → Multiple: 10-20x revenue, 또는 R&D 자산 별도 평가
```

**듀얼 브랜드 전략 = valuation 재평가 트리거**.

## 2. 전략적 전제 — 제약 GLP-1 듀얼 브랜드 모델

### 2.1 검증된 산업 패턴

GLP-1 분자는 **동일한 약리 작용**을 갖지만, 적응증·환자 정체성·임상 가치가 다르므로 글로벌 제약사는 일관되게 듀얼 브랜드 전략을 구사해왔습니다.

| 분자 | 회사 | 당뇨 브랜드 | 비만 브랜드 | 출시 차이 |
|------|------|-----------|-----------|----------|
| **Semaglutide** | Novo Nordisk | **Ozempic** (2017) | **Wegovy** (2021) | 4년 후 비만 적응증 별도 브랜드화 |
| **Tirzepatide** | Eli Lilly | **Mounjaro** (2022) | **Zepbound** (2023) | 1년 만에 동일 분자·동일 용량을 별도 브랜드로 출시 |
| **Liraglutide** | Novo Nordisk | **Victoza** (2010) | **Saxenda** (2014) | 4년 후 고용량 비만 브랜드 출시 |

### 2.2 왜 듀얼 브랜드인가? — 5가지 본질

**① 환자 정체성의 분리**
- 당뇨 환자는 자신을 "당뇨 환자"로 정체화 — 의학적 관리 모드
- 비만 환자는 자신을 "다이어터" 또는 "체중 관리자"로 정체화 — 라이프스타일 모드
- **같은 약·같은 행동도 두 정체성에서 완전히 다른 의미를 가짐**

**② 처방·구매 경로의 분리**
- 당뇨: 내분비내과 → 의사 처방 → 약국
- 비만: 비만 클리닉/일반 의원 → 자가 결정 → 약국
- 채널·메시지·교육 자료가 본질적으로 다름

**③ 보험 적용의 분리**
- 당뇨: 보험 급여, 의료비 재정 정당화 가능
- 비만: 비급여, 자비 부담
- 가격·접근성·마케팅 톤 모두 분리 필요

**④ 임상 근거의 분리**
- 당뇨 임상: HbA1c 감소, TIR 개선, 합병증 예방
- 비만 임상: 체중 감소, 허리둘레, 지방량 변화
- 같은 분자도 적응증별 임상시험을 별도로 수행

**⑤ 마케팅·커뮤니케이션의 분리**
- 당뇨: "혈당 관리, 합병증 예방" — 신중·전문적 톤
- 비만: "변화·자신감·새로운 나" — 활력·동기부여 톤
- 한 브랜드로 두 톤을 동시 구사하면 어느 쪽도 효과 없음

### 2.3 디지털 헬스에서의 함의

**제약이 검증한 모델은 디지털 헬스에서도 동일하게 적용됩니다.** 오히려 디지털 헬스가 더 유리한 점:

| 제약 산업 | 디지털 헬스 (인크레티나) |
|----------|-----------------------|
| 신규 브랜드 = 새 임상시험 ($100M+) | 신규 브랜드 = 캘리브레이션·UI 분기 ($100K~) |
| 듀얼 브랜드 출시 시차 1-4년 | 출시 시차 6-12개월 가능 |
| 의약품 마케팅 규제 강 | 디지털 헬스 마케팅 자유도 높음 |

→ **디지털 헬스에서 듀얼 브랜드 전략은 제약보다 비용·속도 모두 압도적으로 효율적**.

## 3. IMEM 엔진 — 본질과 두 적응증의 분기

### 3.1 IMEM 엔진의 생물학적 본질

IMEM(Incretin Metabolic Efficiency Model)은 **인크레틴 자가분비 효율을 정량화하는 행동 과학 알고리즘**입니다. 핵심은 **3개 계수의 곱**:

```
대사 효율 = α × β × γ
```

| 계수 | 생물학적 의미 | 행동 영역 |
|------|-------------|----------|
| **α (Alpha)** | 일주기 리듬 동조 → 인크레틴 분비 시간 패턴 | 식사·수면·광노출 시간 |
| **β (Beta)** | 영양 시퀀스 → L-cell GLP-1 직접 자극 | 식이 순서·구성 |
| **γ (Gamma)** | 신체 활동 → GLUT4·인슐린 감수성 | 운동 시점·강도 |

**Bio-Sync**는 IMEM의 시간 정렬 모듈로, 각 계수의 골든타임을 사용자에게 카운트다운으로 제시합니다.

### 3.2 두 갈래 작용 — GLP-1의 분기점

GLP-1 분비 후 작용은 **두 개의 축**으로 나뉘며, 어느 축을 메인 내러티브로 가져가느냐가 브랜드의 본질입니다.

```
                   GLP-1 자가분비 (IMEM 엔진의 결과)
                              │
            ┌─────────────────┴─────────────────┐
            ▼                                   ▼
    ① 췌장-혈당 축                       ② 뇌-식욕 축
    인슐린 ↑ + 글루카곤 ↓                시상하부 작용
    β세포 보호                           보상 회로 둔화
    인슐린 감수성 회복                    위 배출 지연
            │                                   │
            ▼                                   ▼
       【당뇨 브랜드】                    【비만 브랜드】
       IMEM-DM                          IMEM (인크레티나 i)
       "혈당 평탄"                       "식욕·체중 다스림"
       HbA1c, TIR, GMI                  체중, 칼로리, 효율
```

### 3.3 같은 계수, 다른 임상 의미

각 IMEM 계수는 두 적응증에서 **다른 임상적 가치와 메시지**를 가집니다.

| 계수 | 비만 브랜드 (인크레티나 i) | **당뇨 브랜드 (IMEM-DM)** |
|------|------------------------|------------------------|
| **α (일주기)** | 대사 리듬 정렬 → 지방 연소 효율 | **새벽현상(Dawn Phenomenon) 억제** + 인슐린 일중 효율 |
| **β (시퀀스)** | 인크레틴 분비 → 포만감 강화 | **식후 혈당 스파이크 평탄화** (PPG variability) |
| **γ (활동)** | 근손실 방지 → 대사율 유지 | **인슐린 감수성 직접 회복** (GLUT4 translocation) |

> **결정적 인사이트**: 비만은 calorie deficit이라는 무딘 메커니즘으로도 가능하지만, **당뇨 관리는 *언제, 무엇을, 어떤 순서로* 먹고 *언제 움직이는지*가 직접 혈당 곡선에 새겨집니다.** IMEM의 시간-순서 정량화는 당뇨에서 의학적 정밀도가 훨씬 높습니다.

### 3.4 Bio-Sync의 분기 — "메타볼릭 스위치" vs "혈당 시계"

**비만 Bio-Sync**: 메타볼릭 스위치
- 14시간 공복 → 지방 대사 전환
- 식후 골든타임 → 글루코스 클리어런스
- 핵심 시간: 19시 마감 (지방 저장 차단)
- 메타포: *"지방을 태우는 시계"*

**당뇨 Bio-Sync**: 혈당 시계 (Glycemic Clock)
- 24시간 혈당 곡선 전체 정렬
- 기상 직후 ~ 새벽까지 모든 시간대에 임상적 의미

| 시간대 | 당뇨 환자에게 일어나는 일 | Bio-Sync DM 개입 |
|--------|------------------------|-----------------|
| 04-08시 | 새벽현상 (코르티솔↑, 성장호르몬↑ → 공복혈당 상승) | 기상-광노출 정렬 → 코르티솔 리셋 |
| 식전 30분 | β세포 다음 식사 준비 | 프리로드 → 1차 인슐린 분비 점화 |
| 식후 30-60분 | 혈당 피크 (정상 140↓ vs 당뇨 200↑) | 식이 시퀀스 + 식후 보행 |
| 식후 2-3시간 | 후기 혈당 변동 | γ 활동 → GLUT4 동원 |
| 19시 이후 | 야간 인슐린 저항성 ↑ | 저녁 마감 → 야간 저혈당 + 새벽현상 동시 차단 |
| 23-04시 | 수면 중 혈당 변동 | 멜라토닌 리듬 → 수면 질 → 인슐린 감수성 |

→ **당뇨 Bio-Sync는 "메타볼릭 스위치"보다 임상적 무게가 깊습니다.** 24시간 전 구간에 개입하기 때문.

### 3.5 IMEM-DM의 추가 입력·출력 (개념 수준)

> ⚠️ 구체적 알고리즘·계수 공식은 본 문서에 포함되지 않습니다. 특허 명세서 참조.

**추가 입력 채널** (개념적으로):
- 혈당 측정값 (자가측정 + CGM 연동)
- HbA1c 분기 입력
- 복약 기록 (메트포민, GLP-1 주사, 인슐린)
- 합병증 모니터링 마커

**확장 출력 지표**:
- 당뇨 표준 임상 지표 (TIR, GMI, GVI 등)
- 약 의존도 변화 추적
- 합병증 위험 점수

## 4. 브랜드 정체성 매트릭스

### 4.1 핵심 대조표

| 차원 | 인크레티나 i (비만) | **IMEM-DM 브랜드 (당뇨)** |
|------|-------------------|------------------------|
| **사용자 정체성** | 다이어터, 체중 관리자 | 당뇨 환자, 전당뇨 관리자 |
| **사용자 동기** | 외모, 자신감, 라이프스타일 | 합병증 회피, 약 줄이기, 건강 회복 |
| **핵심 약속** | "약 없이 식욕·체중을 다스리는 14시간의 리듬" | "주사 없이 평탄한 혈당 곡선을 그리는 24시간의 시계" |
| **성공의 모양** | 우하향 곡선 (체중 ↓) | 평탄한 곡선 (혈당 평탄) |
| **핵심 KPI** | 체중, 칼로리, IMEM 효율 | TIR, HbA1c, GMI, 약 의존도 |
| **시간 메타포** | 메타볼릭 스위치 (14h 공복) | 혈당 시계 (24h 정렬) |
| **GLP-1 비유** | 내 몸의 식욕 조절자 | 내 몸의 인슐린 지휘자 |
| **공포 트리거** | 요요, 정체기, 노화 | 실명, 투석, 절단, 심혈관 |
| **희망 트리거** | 옷 핏, 자신감, 활력 | "약 줄이기", "정상 진단 복귀", 가족과 더 오래 |
| **언어 톤** | 활력·도전·변화 | 신중·평정·통제 |
| **상징색** | 따뜻한 오렌지·골드 (활력) | 차분한 블루·티얼 (신뢰·평정) |
| **타이포그래피** | Friendly sans-serif | Clinical sans-serif + Serif accents |

### 4.2 정체성 분리의 임상·심리학적 근거

**왜 같은 사람이라도 두 브랜드를 동시에 쓰면 안 되는가?**

1. **자기 진단 충돌**: "나는 당뇨 환자이자 다이어터"라는 정체성은 인지적 불일치 발생
2. **메시지 충돌**: 비만 톤("도전!")과 당뇨 톤("신중하게")이 한 화면에 공존하면 어느 쪽도 효과 없음
3. **임상 우선순위 충돌**: 당뇨 환자에게 "단식 14시간"을 무차별 권하면 저혈당 위험
4. **약물 안전성 충돌**: 당뇨 약 복용자에게 비만용 식사 패턴은 의학적 부적합 가능

→ **두 브랜드는 같은 사용자를 둘 다 받지 않는다.** 사용자가 "나는 당뇨 환자다"로 정체화하면 즉시 IMEM-DM 브랜드로 이동.

### 4.3 사용자 여정 분기점

```
사용자 진입 → 자기 진단 (랜딩 페이지)
                    │
        ┌───────────┴───────────┐
        ▼                       ▼
"체중을 줄이고 싶다"        "혈당이 걱정된다"
"건강한 라이프스타일"       "당뇨 진단 받음"
        │                       │
        ▼                       ▼
   인크레티나 i             IMEM-DM 브랜드
   (비만 브랜드)            (당뇨 브랜드)
```

**중간 영역 (전당뇨 + 비만)** 사용자는 어느 브랜드든 진입 가능. 단, 사용 중 우선순위가 명확히 어느 한쪽으로 기울어짐 → 그 시점에 다른 브랜드로 자연 이동 권유 가능.

## 5. 타겟 페르소나 분석

### 5.1 인크레티나 i (비만) — 현재 페르소나

**Primary**: 30-40대 여성, BMI 25-30, 직장인 또는 주부
- 체중 관리 시도 반복, 요요 경험
- 건강·외모 동시 관심
- 디지털 도구 친숙, 텔레그램·카카오 일상 사용
- 가족 식사 책임자 (식이 변화의 게이트키퍼)

**Secondary**: 30-50대 남성, BMI 26-32, 사무직
- 건강검진 적색 신호 (간 수치, 혈압)
- 술자리·외식 빈도 높음
- "정량적 관리" 선호 (점수, 그래프)

### 5.2 IMEM-DM (당뇨) — 신규 페르소나

#### 페르소나 A: "신규 진단 충격형" ⭐ 1순위 타겟

**40-55세, 최근 1년 내 당뇨 진단**
- 의사로부터 "약 시작" 권유 받았거나 막 시작
- "삶이 바뀌었다"는 충격, 변화 의지 최고조
- 정보 갈증: "내가 뭘 어떻게 해야 하는지" 절박하게 검색
- 디지털 도구에 적극 (앱, CGM, 웨어러블)

**핵심 페인**:
- 식단·운동을 "어떻게" 해야 할지 모름
- 의사는 "관리하세요"만 말하고 구체 방법 안 줌
- 당뇨 교육 받았지만 일상에 적용 어려움

**가치 제안**:
> *"당뇨 진단 첫 6개월. 이 시기에 만든 습관이 평생 약 의존도를 결정합니다.
> IMEM-DM은 당신에게 매일·매끼·매시간 무엇을 하면 혈당이 어떻게 반응하는지 알려줍니다."*

#### 페르소나 B: "GLP-1 약물 사용자" ⭐ 2순위 타겟

**30-50세, Wegovy/Ozempic/Mounjaro 사용 중**
- 약효 극대화 + 부작용 관리 + 끊은 후 유지 관심
- 자비 부담 큼 → 효과 극대화 동기 강력
- 라이프스타일 변화에 적극

**핵심 페인**:
- 약 끊으면 요요 두려움
- 부작용(메스꺼움, 탈모) 관리 어려움
- "약 + 라이프스타일"의 시너지 가이드 부재

**가치 제안**:
> *"GLP-1 약을 쓰고 계신가요? IMEM-DM은 약효를 극대화하고,
> 끊은 후에도 자가분비 GLP-1을 유지하는 습관 시스템입니다."*

#### 페르소나 C: "기존 당뇨 환자" (3순위)

**55-70세, 당뇨 5년+, 약 복용 중**
- 관리 피로 (burn-out)
- 합병증 두려움 본격화
- 디지털 친숙도 낮음 → UX 단순화 필수

**핵심 페인**:
- 매일 같은 일 반복, 동기 상실
- 새 정보 받아들이기 어려움
- 가족이 보호자처럼 옆에 붙어있음 (가족 동참 채널 필요)

#### 페르소나 D: "전당뇨 / 가족력" (4순위, 추후)

**35-50세, 공복혈당 100-125 또는 가족 당뇨력**
- 무증상이지만 위험 인지
- 예방 모티베이션 필요
- 비만 브랜드와 일부 겹치는 영역

### 5.3 페르소나별 채널·메시지

| 페르소나 | 진입 채널 | 메시지 톤 | 핵심 메시지 |
|---------|---------|----------|-----------|
| A (신규 진단) | 내과 의원 + 검색광고 ("당뇨 관리법") | 신중·전문 + 따뜻 | "지금이 평생을 결정합니다" |
| B (GLP-1 사용자) | SNS·인플루언서 + 약물 커뮤니티 | 활력·과학 | "약효를 극대화하세요" |
| C (기존 환자) | 가족 추천 + 보험사 제휴 | 따뜻·평정 + 가족 | "오늘도 잘 다스리고 있어요" |
| D (전당뇨) | 검진 결과 알림 + 비만 브랜드에서 전환 | 경각·예방 | "당뇨로 진행을 막을 수 있습니다" |

### 6.1 핵심 슬로건 비교

## 8. 브랜드 네이밍 전략

### 8.1 네이밍 결정 프레임워크

브랜드 네임은 다음 5가지 기준으로 평가:

| 기준 | 가중치 | 설명 |
|------|------|------|
| **임상 신뢰성** | ★★★★★ | 의료진·환자가 신뢰감을 가질 수 있는가 |
| **모브랜드 자산 활용** | ★★★★ | 인크레티나 브랜드 인지를 활용하는가 |
| **차별화 강도** | ★★★★ | 비만 브랜드와 명확히 구분되는가 |
| **글로벌 확장성** | ★★★ | 영문 발음·상표 등록 가능한가 |
| **한국어 발음 친화성** | ★★★ | 한국 시장에서 자연스러운가 |

### 8.2 네이밍 전략 옵션 — 3가지 갈래

#### 전략 A: 패밀리 브랜드 (서브브랜드)
모브랜드를 활용한 네이밍. 인크레티나 패밀리 시스템 구축.

| 후보 | 어원 | 장점 | 단점 |
|------|------|------|------|
| **인크레티나 G** | G = Glycemic | 모브랜드 100% 활용, 시스템 명확 (i / G) | 차별화 약함 |
| **인크레티나 dia** | dia = diabetes | 직관적, 패밀리감 | 영문 어색 |
| **IncretinA D** | D = Diabetes | 글로벌 확장 용이 | 한국어 어색 |
| **IncretinA Plus** | 약 + IMEM 시너지 강조 | 약물 사용자 타겟 명확 | 모호함 |

#### 전략 B: 독립 브랜드 (제약 GLP-1 패턴)
완전히 다른 이름. Mounjaro/Zepbound 패턴.

| 후보 | 어원 | 장점 | 단점 |
|------|------|------|------|
| **글루세라 (Glucera)** | Glucose + cera (밀랍, 평탄) | 의약품 신뢰감, 부드러운 발음, 영한 자연 | 모브랜드 자산 활용 X |
| **플라나 (Plana)** | Plain + 평탄, 라틴어 평면 | 글로벌 확장, "평탄"이라는 핵심 가치 직격 | 한국 인지도 0에서 시작 |
| **글루코다 (Glucoda)** | Gluco + Coda (악장 마침) | 종결·완결의 음악적 메타포 | 다소 추상적 |
| **이쿼리 (Equari)** | Equilibrium 균형 | 평정·균형 메시지 | 의료적 강도 약함 |
| **스테디 (Steady)** | 안정 | 직관적 | 흔한 단어 |
| **인슐로 (Insul-o)** | 인슐린 + 원형 (균형) | 한국적 친근, 인슐린 강조 | 글로벌 어색 |

#### 전략 C: 메커니즘 강조형
GLP-1 메커니즘을 직접 차용한 네이밍.

| 후보 | 어원 | 장점 | 단점 |
|------|------|------|------|
| **GLP-Wake** | GLP-1 깨우기 | 메커니즘 직관 | 영문 전용 |
| **AutoGLP** | Auto + GLP | 자가분비 강조 | 다소 기계적 |
| **Endogenex** | Endogenous (내인성) | 과학적 톤 | 발음 어려움 |

### 8.3 추천 Top 3 (제안)

#### 🥇 1순위: **글루세라 (Glucera)**
- **이유**: 임상 신뢰감 + 부드러운 발음 + 영한 모두 자연 + "Glucose가 평탄해지는 곳"이라는 의미가 자연 연상
- **태그라인 예시**: *"글루세라 — 내 몸의 인슐린 지휘자"*
- **단점 대응**: 모브랜드 자산은 패키지 디자인·"Powered by IncretinA" 표기로 활용

#### 🥈 2순위: **인크레티나 G**
- **이유**: 모브랜드 인지 100% 활용, 패밀리 시스템 자체가 IR 스토리 ("우리는 인크레티나 패밀리를 만든다")
- **태그라인 예시**: *"인크레티나 G — 혈당의 평탄을 위한 GLP-1"*
- **단점 대응**: "i / G" 시스템을 마케팅·패키지·UI 모두에 일관 적용해 차별화 강화

#### 🥉 3순위: **플라나 (Plana)**
- **이유**: "Flatline이 좋은 것"이라는 당뇨 특유의 가치를 한 단어로 직격, 글로벌 확장 유리
- **태그라인 예시**: *"플라나 — 평탄한 혈당, 평탄한 하루"*
- **단점 대응**: 초기 인지도 구축에 마케팅 자원 추가 필요

### 8.4 의사결정 권장

**제 권장**: **인크레티나 G (1차)** + **글로벌 진출 시점 글루세라로 리브랜딩 검토 (2차)**

이유:
1. 시드/시리즈A 단계에서는 모브랜드 자산 활용이 자금 효율적
2. "패밀리 브랜드 시스템" 자체가 IR 스토리 (단일 제품 → 플랫폼 인식)
3. 한국 시장 검증 후 글로벌 진출 시점에 독립 브랜드로 전환 가능
4. 제약 패턴(예: Saxenda 출시 후에도 Liraglutide 동일 약제 인식 유지)과 일치

### 8.5 상표 출원 우선순위

6개월 내:     IncretinA G (US, EU, 일본)
12개월 내:    글루세라, 플라나 (방어용 출원)

## 10. 기술 자산 공유 아키텍처

### 10.1 공유 vs 분기 매트릭스

| 컴포넌트 | 공유 / 분기 | 설명 |
|---------|----------|------|
| IMEM Core (α/β/γ 골격) | **공유** | 공통 패키지 |
| Bio-Sync 시간 정렬 | **공유** | 같은 알고리즘 |
| Calibration (계수 가중치) | **분기** | 비만용 / 당뇨용 별도 |
| Routine 카탈로그 | **공유 + 확장** | 공통 10개 + 당뇨 추가 |
| Risk 카탈로그 | **공유 + 확장** | 공통 8개 + 당뇨 신규 5개+ |
| Narrator (해석 레이어) | **분기** | weight-narrator / glycemic-narrator |
| Telegram Bot (백엔드) | **공유** | 같은 Fly.io 인프라 |
| 텔레그램 봇 페르소나 | **분기** | 비만 3종 + 당뇨 4종 |
| Firestore 스키마 | **공유 + 확장** | 공통 필드 + DM 전용 컬렉션 |
| 앱 PWA | **분기** | 별도 도메인·UI |
| 인증·결제 시스템 | **공유** | 공통 IncretinA ID |

### 10.2 코드 재사용율 추정

```
공통 코어 (imem-core, bot 백엔드):  ~75% 재사용
브랜드별 분기 (UI, narrator, calibration):  ~25% 별도 개발
```

### 10.3 IncretinA 공통 ID 시스템

두 브랜드 사용자가 모두 **단일 IncretinA ID**로 통합:
- 한 사람이 비만 브랜드 → 당뇨 브랜드 이동 시 데이터 연속성
- 가족 구성원 간 연결 가능 (당뇨 환자 + 가족 다이어터)
- 향후 추가 적응증(NAFLD, PCOS 등) 추가 시 자연스러운 확장

### 10.4 신규 적응증 확장 청사진

같은 1엔진 2적응증 패턴은 다음 영역으로도 확장 가능:

| 적응증 | 시장 규모 | IMEM 확장 포인트 |
|--------|---------|---------------|
| **비알코올성 지방간(NAFLD)** | 글로벌 $5B+ | 인슐린 감수성·간 대사 |
| **PCOS (다낭성 난소 증후군)** | 글로벌 $3B+ | 인슐린 저항·호르몬 리듬 |
| **노인 근감소증(Sarcopenia)** | 글로벌 $7B+ | γ 강화 + 단백질 시퀀스 |
| **수술 전후 대사 관리** | B2B 의료기관 | α/β 정밀 정렬 |
| **임신성 당뇨** | 한국 연 5만+ 임산부 | β + 안전 캘리브레이션 |

→ **각 적응증마다 별도 브랜드 출시 가능** (브랜드 시리즈 확장).

### 10.5 기술 부채 방지 원칙

1. **imem-core는 절대 단일 적응증 가정으로 작성하지 않는다** — 모든 함수가 calibration profile을 인자로 받음
2. **narrator는 항상 분리** — 코어 로직과 표현이 절대 섞이지 않음
3. **신규 계수 추가는 코어에서** — 적응증 코드에서 자체 계수 추가 금지
4. **Firestore 스키마는 versioning** — 적응증 추가 시 마이그레이션 경로 보장

## 11. 규제 & 임상 경로

### 11.1 규제 카테고리 비교

| 항목 | 인크레티나 i | IMEM-DM |
|------|------------|---------|
| **현 규제 분류** | 웰니스 앱 (규제 외) | 웰니스 앱 (초기) → 디지털 치료기기(SaMD) (장기) |
| **식약처 인증** | 불필요 | 단계적 (Class I → Class II) |
| **개인정보 등급** | 일반 개인정보 | **민감 의료정보** — 별도 동의·암호화 |
| **광고 규제** | 표시광고법 | + 의료기기 광고 사전심의 (Class II 시) |

### 11.2 IMEM-DM 단계별 규제 경로

**Phase 1: 웰니스 앱 (출시 ~+12개월)**
- 의료기기 인증 없이 출시 가능
- "관리·기록·라이프스타일 보조" 표현 고수
- 진단·치료·처방 표현 절대 금지
- 의사 권유는 "참고 자료"로만 활용

**Phase 2: SaMD Class I (+12 ~ +24개월)**
- 식약처 의료기기 신고 (Class I — 신고만 가능)
- HbA1c·TIR 데이터 기록 기능 의료기기로 등록
- 보험 적용 사전 작업

**Phase 3: SaMD Class II (+24 ~ +36개월)**
- 식약처 의료기기 인증 (Class II — 임상시험 필요)
- 처방형 디지털 치료기기 진입
- 보험 급여 신청 가능

### 11.3 임상 근거 축적 전략

| 단계 | 임상 활동 | 목적 |
|-----|---------|------|
| **+6개월** | 대학병원 내분비내과 1-2곳 협업 MOU | 임상 자문, 추후 RCT 기반 |
| **+12개월** | 파일럿 IRB (소규모 단일군 관찰) | 안전성·유효성 초기 신호 |
| **+18개월** | RCT 디자인·IRB 승인 | 무작위배정 비교 임상 |
| **+24개월** | RCT 실시 (n=100-200, 12주) | HbA1c·TIR 변화 |
| **+30개월** | 결과 발표 (학술지) | 마케팅·규제·보험 동시 활용 |
| **+36개월** | RWE (Real World Evidence) 축적 | DTx 인증 패키지 |

### 11.4 임상 파트너십 우선순위

**1순위 협업 대상**:
- 서울대병원 내분비내과 (당뇨 연구 권위)
- 강북삼성병원 당뇨센터 (코호트 데이터 보유)

**2순위**:
- 한양대병원, 가천길병원 (지역 거점)
- 당뇨 협회·당뇨 교육사 협회

**3순위 (B2B 채널 확장)**:
- 카카오헬스케어 (CGM 협력)
- 보험사 (삼성생명, 현대해상)

### 13.2 차별화 포인트 우선순위

1. **메커니즘 기반 정량 엔진** — IMEM α/β/γ는 경쟁사가 모방 불가
2. **텔레그램 AI 코치** — 일상 친근 채널, 진입 장벽 낮음
3. **인크레티나 패밀리** — 비만·당뇨 통합 LTV
4. **임상 근거 + DTx 진입** — 장기 경쟁 우위
5. **한국 시장 특화** — 한국어 UX, 한국 의료 환경 이해
